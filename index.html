Rock Paper Scissors - Developer Console Game

Description:

This is a 1-player game of 5 rounds of Rock Paper Scissors against an automated opponent.

The game takes place in the Google Chrome Developer Console (or any JavaScript-compatible console).

How to Play:

  1. Open the Chrome Developer Console.
    • Shortcuts (pick one):
    • F12
    • Right-click on page; Choose 'Inspect'.
  2. Type or copy-paste the following in the 'Console' and press Return/Enter:
    • playGame()
  3. Follow the instructions in the prompt and look at the console to see the results.
